# ViniCogs

Production-grade cogs for Red-DiscordBot, built for performance, automation, and next-level Discord experiences.

---

## Dynamic Music Presence for Red

Show the currently playing track directly in your bot's Discord status.
Built for Red Audio + Lavalink.

---

## 🎧 Features

* Dynamic "Now Playing" status based on the current track
* Seamless integration with Red Audio + Lavalink
* Lightweight and optimized for low-resource environments
* Easy configuration via in-bot commands
* Designed for scalability and real-world usage

---

## 🚀 Installation

```text
[p]load downloader
[p]repo add vinicogs https://github.com/Vinizera-ai/ViniCogs
[p]cog install vinicogs musicpresence
[p]load musicpresence
```

---

## ⚙️ Configuration

```text
[p]mpstatus prefix Now playing:
[p]mpstatus idle listening Music 24/7
```

---

## 📦 Current Cogs

### 🎵 musicpresence

Displays the currently playing track directly in the bot's Discord status.

**Example:**

```
Listening to Now playing: Logic - Ballin (ft. Castro)
```
